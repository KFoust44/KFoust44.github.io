Terraria build guide!

Home page

With Terraria 1.4 AKA "Journeys End" finally out I found it appropriate to make a build guide for myself and others that enjoy playing this game as much as I do.

All pictures on this website are screenshots of Terraria taken by me.

To be clear this isn't a playthrough guide, it's simply a guide that tells you what to look for gear-wise.

To start there are four main classes, each one has it's own unique fighting style that fits different types of people. Some of the items that you will use will be interchangable between all classes.

Interchangable Gear

Armor

There is two types of armor, Class specific and Base armor. If you are brave enough you can go straight for Class specific, but that's difficult and having base armor is quite useful.

The order goes: Wood Armor, Cactus Armor, Copper Armor, Tin Armor, Iron/Lead Armor, Silver/Tungsten Armor, Gold/Platinum Armor.
